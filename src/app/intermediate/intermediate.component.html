Instrucciones

El objetivo de esta prueba es realizar la creación y lectura de un entidad (empleado).

Utilice el siguiente endpoint para leer todas las entidades usando una solicitud (Request) GET: http://dummy.restapiexample.com/api/v1/employees

Se recomienda usar Reactive Forms, pero no es estrictamente necesario ;)

La entidad es representada en JSON de la siguiente forma: 
{{{
    id: '1',
    employee_name: 'Nombre',
    employee_salary: '100',
    employee_age: '100',
    profile_image: ''
  } | json}}

Para crear un nuevo empleado envie una solicitud (Request) POST el siguiente endpoint: http://dummy.restapiexample.com/api/v1/create

Utilice el siguiente formulario para capturar los datos y realice la solicitud POST al hacer click en el boton:

Id
Nombre
Salario

Al hacer click en el siguiente botón se deben leer los datos de la API y mostrarlos en la tabla
