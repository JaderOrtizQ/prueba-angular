Prueba Básica

El objetivo de esta prueba es usar una API publica para obtener información y mostrarla en una tabla.

La siguiente tabla debe mostrar la lista de departamentos y ciudades despues de invocar el API publica: https://www.datos.gov.co/resource/xdk5-pm3f.json

Si lo desea, podrá encontrar la documentación del API en el siguiente link: https://dev.socrata.com/foundry/www.datos.gov.co/xdk5-pm3f

Utilice la clase BasicService para invocar el API y retornar los resultados al componente que va a mostrar la información en la siguiente tabla:
